# Wheel of Fortune

A simple browser-based wheel application built with HTML5 Canvas and JavaScript.

## Features
- Spin an interactive wheel with names or Yes/No options
- Add names via the input field (each name is added twice)
- Remove names by clicking on them in the wheel (only when the wheel is not spinning)
- Toggle button to add or remove three Yes and three No entries
- Minimum spin duration of 5 seconds with realistic deceleration
- Responsive design for both mobile and desktop
